# electron-notepad

## Iniciar o projeto 
<div align="justify">Para iniciar o projeto, clone o repositório em sua máquina, entre dentro da pasta e rode o comando: 
<br>
<code> npm install </code>
<br> 
<br> Agora, para abrir o notepad, rode o comando: 
<br>
<code> npm start </code> 
</div>
<br>
